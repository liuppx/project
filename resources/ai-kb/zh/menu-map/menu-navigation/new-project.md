# 新建项目入口在哪

## 路径
- 桌面端：右上角「+」下拉 →「新建项目」
- 桌面端：左侧栏「应用」→「创建项目」卡片
- 桌面端：头像下拉菜单（管理员）→「所有项目」→ 右上「+」
- 移动端：底部 Tabbar「应用」→「创建项目」卡片
- 移动端：项目 Tab 右上角「+」
- 桌面快捷键：Cmd/Ctrl + B

## 权限要求
- end-user 默认可建，由系统配置「project_create」开关控制
- 不需要管理员权限
