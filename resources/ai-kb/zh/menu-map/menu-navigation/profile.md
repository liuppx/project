# 个人资料入口在哪

## 路径
- 桌面端：右上角头像下拉 →「个人设置」→「个人设置」（资料）子页
- 桌面端 URL：`/manage/setting/personal`
- 桌面端：直接点左侧栏顶部自己的头像也可触发头像菜单
- 移动端：底部 Tabbar「我的」→「设置」→ 资料
- 快捷键：无

## 可编辑字段
- 头像（点击头像上传图片）
- 昵称
- 职位 / 性别 / 生日（视系统开关）
- 个人简介

## 权限要求
- 所有登录用户可编辑自己的资料
- 部门归属由管理员维护，本页不可改
