# 仪表盘入口在哪

## 路径
- 桌面端：左侧栏顶部「仪表盘」（路由 `/manage/dashboard`）
- 桌面端：登录后多数账号默认落到仪表盘
- 移动端：底部 Tabbar「仪表盘」Tab
- 桌面 URL：`https://<域名>/manage/dashboard`
- 快捷键：无

## 看到什么
- 顶部三个数字卡片：今日到期、超期、待完成
- 下方四个分组：今日 / 超期 / 待完成 / 协助
- 「我的项目」卡片

## 权限要求
- 所有登录用户可见
- 内容是当前用户私有视角
