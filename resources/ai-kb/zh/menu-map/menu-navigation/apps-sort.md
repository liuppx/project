# 应用排序入口在哪

## 路径
- 桌面端：左侧栏「应用」→ 应用中心右上角「排序」/「编辑顺序」按钮
- 桌面端：进入排序模式后拖动卡片调整顺序，再点「保存」
- 移动端：底部 Tabbar「应用」→ 同样有「排序」按钮
- 快捷键：无

## 怎么排
1. 进入排序模式，卡片出现拖拽手柄
2. 拖动卡片到目标位置
3. 「常用应用」与「管理员应用」按分组独立排序
4. 保存后立即生效

## 权限要求
- 所有登录用户可改自己的排序
- 个人配置，互不影响

## 不在排序里的项
- 已卸载的插件卡片：先到 [[menu-navigation.appstore.menu-map]] 安装
- 隐藏的卡片：通常因角色 / 权限不足，无法手动显隐
